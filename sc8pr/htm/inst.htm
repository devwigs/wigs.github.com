Installation

Please ensure that you have Python 3 installed before proceeding. Python versions 3.7 through 3.13 are suppported.

Next, open your terminal program (e.g. Command Prompt or Windows PowerShell). If you are installing sc8pr for all users you will likely need to run the terminal as an administrator. Make sure you have the latest versions of pip and setuptools by running the following command... 

pip3 install -U pip setuptools

On Windows, you may need to replace the command pip3 at the beginning of the line by py -3 -m pip or specify a particular version of Python if more than one is installed: py -3.10 -m pip. If this still doesn’t work, locate the folder where Python is installed and use its Scripts folder as your current directory...

cd "C:/Program Files/Python310/Scripts"
./pip3 install -U pip setuptools

[Really old versions of Windows may require you to use a backslash instead of a forward slash.]

Finally, install sc8pr. This should check if pygame and other dependencies are already installed, and install them from PyPI if neceessary.

To install all features including those with extra dependencies:

pip3 install -U sc8prx

To install core sc8pr features that depend only on pygame:

pip3 install -U sc8pr

If pygame is not already installed and pip cannot locate a build for your platform and Python version, you will need to install it manually before using sc8pr. Please visit the pygame website if this happens.
